Medical Clinic Time Manager
A cross-platform desktop application built with C++ and Qt Framework.

Core Features:
Offline-First Architecture: Uses an SQLite backend for data persistence without cloud dependencies.

Automated Payroll Reporting: Generates logic-based reports from time-tracking data.

Data Integrity: Implements custom database schema to handle vacations, breaks, and working hours.

Technical Stack:
Language: C++17/20

Framework: Qt (Widgets/Signals & Slots)

Database: SQLite
